Je n'ai pas de binôme pour ce projet.

J'ai choisi le langage python, car c'est le langage que nous avons, le plus vu cette année.

L'ensemble des codes demandés ont été traités dans l'ensemble, en chiffrage et déchiffrage :

1-Vigenere
2-Hill
3-Transposition rectangulaire
4-DES

L'ensemble des explications se trouve en commentaire ou dans les "print". J'ai détaillé au maximum ce que 
fait mon code en console, afin d'avoir le maximum de visu.
Pour avoir des informations sur comment exécuter le code, il suffit de d'exécuter le code sans paramètre.
Exemple : MonProgramme.py (Entrer) et les instructions seront affichées.

Il est nécessaire de mettre entre guillemets " " ou entre quotes ' ' , la clé si cette dernière possède 
des caractères qui sont ni des lettres ni des chiffres (sachant que nous considérons les caractères
ASCII [32, 126]), afin que python puisse considérer que c'est une chaine de caractère et non un # 
par exemple si la chaîne est constitué d'un # (cela est précisé dans les instructions).
Ce qui est valable pour Vigenère et la Transposition rectangulaire, le plus simple reste de toujours 
mettre les guillemets.

Pour le DES, il ne prend en entrée qu'une chaîne hexadécimal de taille 16 (64 bits), sans ajouter le "0x",
pour le chiffrage comme le déchiffrage.
